Example of custom and impure pipe.

fetchJson Pipe

  • Take url as input
  • Return result of url after http call

    • In below example we used custom pipe 'fetchJson'. It will return json object. And to parse that json object we used default pipe 'Json'

    "url | fetchJson | json"

    {{ "http://ip-api.com/json/24.48.0." + i | fetchJson | json }}

    Example default pure pipe 'Date pipe'.

    Date Pipe

  • Formats a date value according to locale rules.
  • Take values as string | number | Date

    • In below example we used custom pipe 'fetchJson'. It will return json object. And to parse that json object we used default pipe 'Json'

    "value_expression | date [ : format [ : timezone [ : locale ] ] ]"

    {{ dateObj }}
    dateObj | date -> {{ dateObj | date }}
    dateObj | date: "medium" -> {{ dateObj | date: "medium" }}
    dateObj | date: "shortTime" -> {{ dateObj | date: "shortTime" }}
    dateObj | date: "longDate" -> {{ dateObj | date: "longDate" }}
